# Club Activity Registration System
A PHP & MySQL web application for students to manage and join club activities.

## Features
- User registration, login & logout with password hashing
- Activity display with images and details
- Register/unregister activities (prevent duplicate sign-ups)
- Admin-only activity management (delete, upload images)
- Search registrations by name or activity
- Statistics bar chart using Chart.js

## Technologies
- PHP (PDO)
- MySQL
- HTML/CSS
- JavaScript / Chart.js

## How to Run
1.  Import `club_system.sql` into your MySQL database
2.  Update database credentials in `config.php`
3.  Place all files in your web server (e.g., XAMPP)
4.  Open `dashboard.php` in browser
